Abstract

Click to Expand

This aims to develop an accurate body measurement system using a hybrid approach that combines Phase 3D and HRNet.

By capturing three still images (front, side, back) using a webcam, key body landmarks are detected, and the distances between these points are calculated for precise measurement.

Project Features

See Features
  • Real-time body measurement analysis.
  • Hybrid AI approach using Phase 3D & HRNet.
  • Accurate landmark detection for precise body dimensions.
  • Compatible with standard webcams for easy deployment.
